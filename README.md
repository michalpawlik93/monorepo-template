# Node.js Template

## Struktura projektu

Repozytorium zawiera obecnie jeden folder:

- **`backend/`** - monorepo NX z aplikacją backendową

## Quick Start

Przejdź do folderu `backend/` i zapoznaj się z [README.md](backend/README.md).

Aplikację można uruchamiać lokalnie za pomocą **Makefile**:

```powershell
cd backend
make setup
make serve
```
